Customize Your Maintenance Pages

Each template folder contains index.html, styles.css, script.js, logo.svg, and favicon.svg. All are pure HTML/CSS/JS, no frameworks.

Change the Logo

  1. Replace logo.svg with your own SVG, or edit the file directly.
  2. Keep the width/height reasonable (e.g., 140Ã—40). The nav will scale automatically.

Update the Favicon

  1. Replace favicon.svg in the same folder.
  2. Favicons are referenced in index.html with <link rel="icon" type="image/svg+xml" href="favicon.svg">.

Change the Countdown Target

  1. Open index.html.
  2. Find the script that sets window.MAINTENANCE_DEADLINE.
  3. Set it to an ISO date, e.g., 2025-12-31T23:59:59Z.

Alternatively, in script.js replace how target is computed.

Reset the Countdown

Set window.MAINTENANCE_DEADLINE to a future date/time. If the timer reaches zero, it will stop at 00:00:00:00.

Change Colors and Background

Edit CSS variables at the top of styles.css:

:root {
  --color-primary: #7C3AED;
  --color-secondary: #06B6D4;
  --color-accent: #F59E0B;
  --bg-gradient: linear-gradient(135deg,#0F172A,#1E293B);
}

Change Title, Paragraphs, Icons, Elements

Open index.html and update the markup: the title is inside <h1 class="title">, the paragraph uses .lead, and features are a list inside #overview. Icons use CDN classes (Font Awesome / Boxicons / Bootstrap Icons).

Background Animations

Orbs are controlled by .orb--1 and .orb--2. Small floating particles are created in script.js. Adjust counts, sizes, and speeds to your taste.

Mobile Menu

The hamburger toggles the .menu__list. It is fully keyboard-accessible.
